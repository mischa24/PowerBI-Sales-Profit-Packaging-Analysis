# PowerBI Sales, Profit & Packaging Analysis

An interactive Power BI dashboard for comprehensive analysis of:
- 📈 Sales performance  
- 💰 Profitability metrics  
- 📦 Packaging efficiency  
- 🚚 Shipping cost breakdown  
- 🌍 Regional sales trends
